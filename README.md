# Personal Portfolio Website

This repository contains my personal portfolio website, created to showcase my projects, skills, and professional information. The website serves as an online portfolio and represents my learning journey and experience in frontend web development.

The portfolio is designed with a focus on simplicity, clarity, and responsiveness, ensuring a smooth user experience across different devices and screen sizes.

---

## Purpose of the Project

The main goals of this project are:

- To showcase my frontend development skills
- To present my projects in a structured and organized manner
- To create a professional online presence
- To practice building responsive and user-friendly web interfaces

---

## Technologies Used

- **HTML5** – for structuring the content
- **CSS3** – for styling and layout
- **JavaScript** – for interactivity and dynamic behavior

---

## Project Structure

The project follows a simple and easy-to-understand structure:

portfolio/
│
├── index.html # Main HTML file
├── style.css # Stylesheet for the website
├── script.js # JavaScript for interactivity
└── assets/ # Images and other static resources

---

## Features

- Responsive layout suitable for different screen sizes
- Well-organized sections such as About, Skills, Projects, and Contact
- Clean and minimal user interface
- Easy to customize and extend

### Option 1: Open Directly

1. Download or clone the repository
2. Open the `index.html` file in any modern web browser

### Option 2: Run on Localhost

1. Navigate to the project directory
2. Run the following command:

   ```bash
